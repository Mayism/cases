# 会员一码通 需求PRD

> 版本：1.0.0 | 平台：HarmonyOS | 更新日期：2026-05-06

## 1. 业务场景
购物商场会员系统需要提供一个会员一码通功能，作为线下会员服务的核心入口。会员在商场内消费时，出示一码通二维码，收银员扫描后即可完成会员识别、积分累积和电子钱包支付。该功能位于应用底部Tab栏的中间位置，是连接线上会员系统和线下消费场景的关键桥梁。

## 2. 业务目标
- 实现会员线下身份识别，通过动态二维码替代实体会员卡
- 整合会员识别、积分累积、电子钱包支付三合一功能入口
- 提供会员签到激励机制，提升用户活跃度
- 提供会员手册查看入口，帮助用户了解会员权益和规则

## 3. 功能需求

### 3.1 一码通Tab入口 [P0]
- 在底部Tab栏中间位置展示"一码通"Tab
- Tab图标突出显示（比其他Tab更大，向上偏移）
- 已登录用户点击直接跳转至一码通页面
- 未登录用户点击触发华为账号手机号授权，授权成功后跳转至一码通页面

### 3.2 动态二维码展示 [P0]
- 页面中央展示会员二维码，尺寸206x206像素
- 二维码中心叠加应用Logo（56像素）
- 二维码内容为 `用户ID + 时间戳`，确保每次生成的二维码唯一
- 二维码外围容器240x240，白色背景，圆角12，带阴影效果

### 3.3 二维码自动刷新 [P0]
- 二维码每5分钟（300秒）自动刷新一次
- 自动刷新时重新生成二维码内容（用户ID + 新时间戳）
- 页面显示时启动刷新定时器，页面隐藏时清除定时器

### 3.4 手动刷新 [P0]
- 提供手动刷新按钮，包含刷新图标和"刷新"文字
- 点击刷新按钮时刷新图标执行360度旋转动画（EaseInOut, 200ms）
- 刷新后二维码内容更新，300秒倒计时重新开始

### 3.5 签到功能 [P1]
- 页面右上角（用户信息区域右侧）提供签到按钮
- 未签到时显示"签到"按钮，可点击
- 签到成功后按钮文字变为"已签到"，按钮禁用
- 签到成功后用户积分增加10分
- 签到成功后显示Toast提示"签到成功~"

### 3.6 用户信息展示 [P1]
- 页面顶部展示用户头像和手机号（已登录时）
- 未登录时显示登录引导

### 3.7 会员手册 [P1]
- 在"我的"页面快捷入口区域提供"会员手册"入口
- 点击后跳转至会员手册页面
- 会员手册页面展示会员章程内容，包括：会员开通、会员说明、会员资格

## 5. 交互流程

### 5.1 用户点击一码通Tab [P0]
1. 用户点击底部Tab栏"一码通"图标
2. 系统检查用户登录状态（isRelative）
3. 若已登录 → 直接跳转至 MemberShipPage
4. 若未登录 → 显示 FunctionalButton 授权按钮
5. 用户授权手机号 → 成功后跳转至 MemberShipPage
6. 授权失败 → 调用 LoginUtils.onBindFail 处理

### 5.2 二维码自动刷新流程 [P0]
1. 页面显示（onShown）→ 调用 refreshTime()
2. refreshTime() → 执行刷新图标旋转动画
3. 清除旧定时器 → 生成新二维码内容（用户ID + 当前时间戳）
4. 设置 count = 300（秒）
5. 启动 setInterval 每秒递减 count
6. count 降至 0 时清除定时器
7. 页面隐藏（onHidden）→ 清除定时器

### 5.3 手动刷新流程 [P0]
1. 用户点击刷新按钮
2. 调用 refreshTime() → 与自动刷新相同的逻辑
3. 刷新图标旋转动画执行

### 5.4 签到流程 [P1]
1. 用户点击签到按钮
2. globalPointInfo.isCheckIn 设为 true
3. globalPointInfo.myPoints 增加 10
4. 按钮文字变为"已签到"，按钮禁用
5. 显示 Toast "签到成功~"

### 5.5 查看会员手册流程 [P1]
1. 用户在"我的"页面点击"会员手册"快捷入口
2. 通过 RouterUtil.pushPathByName 跳转至 MemberShipManualPage
3. 页面展示会员章程内容

## 6. 功能边界与限制条件

- [P0] 二维码内容格式为 `用户ID + 时间戳`，为纯本地生成，不涉及服务端交互
- [P0] 二维码刷新间隔为固定300秒，不可配置
- [P0] 未登录用户必须完成华为账号手机号授权才能进入一码通页面
- [P1] 签到功能为每日一次，签到状态存储在 GlobalPointUtils 全局状态中
- [P1] 签到积分固定为10分
- [P1] 会员手册内容为静态文本，不从服务端获取

## 7. 验收标准与成功指标
- [P0] 功能完整性：一码通Tab可正常点击并跳转至二维码展示页面
- [P0] 二维码正确生成并展示，中心带有应用Logo
- [P0] 自动刷新和手动刷新功能正常工作
- [P0] 已登录用户点击Tab直接进入，未登录用户触发授权流程
- [P0] 会员手册页面可通过"我的"页面入口正常访问
- [P0] 编译成功：`hvigorw --mode module -p product=default assembleHap --analyze=normal --parallel --incremental --daemon`
- [P1] 签到功能正常，积分正确增加
- [P1] 二维码刷新时图标旋转动画流畅
