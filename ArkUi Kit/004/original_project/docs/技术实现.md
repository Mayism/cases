# 会员一码通 技术实现

> 版本：1.0.0 | 平台：HarmonyOS | 更新日期：2026-05-06

## 1. 技术架构

### 整体架构
会员一码通功能采用模块化架构，位于 `features/business_mine` HAR 模块中，与应用外壳通过路由导航交互。

```
products/phone（应用外壳）
├── IndexPage.ets                    主容器，包含 Tabs（首页/一码通/我的）
├── CustomTabBar.ets                 底部Tab栏组件
├── Constants.ets                    Tab配置常量
└── HomePage.ets                     首页

features/business_mine（会员模块）
├── pages/
│   ├── MemberShipPage.ets           会员一码通主页面
│   ├── MemberShipManualPage.ets     会员手册页面
│   └── MinePage.ets                 "我的"页面
├── viewmodel/
│   └── MinePageViewModel.ets        "我的"页面ViewModel
└── resources/
    └── profile/route_map_mine.json  路由表

commons/lib_common（公共模块）
├── constants/RouterMap.ets          路由名称枚举
├── models/TabBarModel.ets           Tab类型枚举
├── components/
│   ├── LoginComp.ets                登录组件（LoginButtonIcon/LoginButtonText）
│   ├── PageHeaderComp.ets           页面头部组件
│   └── NavHeaderBar.ets             导航栏组件
└── utils/
    ├── RouterUtil.ets               路由工具类
    └── LoginUtils.ets               登录工具类

components/module_points（积分模块）
└── GlobalPointUtils                 全局积分管理器
```

### 组件层级
```
IndexPage (@Provider currentIndex)
├── Tabs
│   ├── TabContent[0] → HomePageBuilder
│   ├── TabContent[1] → 空页面（一码通使用 push 跳转而非 TabContent 渲染）
│   └── TabContent[2] → MinePageBuilder
└── CustomTabBar (@Consumer currentIndex)
    └── ForEach → TabItem
        └── if ONE_CODE_PASS → LoginButtonIcon / Image
        └── else → Image + Text

MemberShipPage (NavDestination)
├── NavHeaderBar
├── PageHeaderComp (用户信息 + 签到按钮)
├── QRCode + Image (Logo叠加)
├── Text (提示信息)
└── Row (刷新按钮)

MemberShipManualPage (NavDestination)
├── NavHeaderBar
└── Column (会员章程内容)
```

### 状态管理方案
- **跨组件通信**：@Provider/@Consumer 模式传递 currentIndex（Tab索引）
- **全局状态**：AppStorageV2.connect 连接 UserInfoModel、GlobalPointUtils
- **局部状态**：@Local 装饰器管理 qrcodeUrl、count、timer、angle 等
- **ViewModel**：@ObservedV2/@Trace 装饰器管理 MinePageViewModel

## 2. 页面组件实现

### MemberShipPage
- **文件路径**：`features/business_mine/src/main/ets/pages/MemberShipPage.ets`
- **组件装饰器**：@ComponentV2
- **Builder 导出**：`MemberShipPageBuilder`

**状态管理**：
```typescript
@Local userInfo: UserInfoModel = AppStorageV2.connect(UserInfoModel, () => new UserInfoModel())!;
@Local globalPointInfo: GlobalPointUtils = AppStorageV2.connect(GlobalPointUtils, () => new GlobalPointUtils())!;
@Local qrcodeUrl: string = this.userInfo.userInfo.id + new Date().getTime()
@Local count: number = 20
@Local timer: number = 0
@Local angle: number = 0;
```

**UI结构**：
- NavDestination（hideTitleBar: true）
  - Stack（背景图 + 内容）
    - Image（main_bg 全屏背景）
    - Column
      - NavHeaderBar（title: '会员一码通'）
      - PageHeaderComp（用户信息 + rightBtnBuilder：签到按钮）
      - Stack（二维码容器 240x240 白色圆角12阴影）
        - QRCode（qrcodeUrl 206x206）
        - Row > Image（ic_logo 56px 居中叠加）
      - Column（提示文字）
      - Row（刷新按钮，灰色背景圆角20）

**事件处理**：
- `onShown`：调用 refreshTime() 启动二维码刷新
- `onHidden`：clearInterval 清除定时器
- 刷新按钮 onClick：调用 refreshTime()
- 签到按钮 onClick：更新 globalPointInfo.isCheckIn 和 myPoints

**关键方法**：
```typescript
refreshTime() {
  this.rotateRefreshIcon();           // 旋转动画
  clearInterval(this.timer);           // 清除旧定时器
  this.qrcodeUrl = this.userInfo.userInfo.id + new Date().getTime(); // 生成新二维码
  this.count = 300;                    // 300秒倒计时
  this.timer = setInterval(() => {
    if (this.count <= 0) { clearInterval(this.timer); return; }
    this.count--;
  }, 1000);
}

rotateRefreshIcon() {
  this.getUIContext().animateTo({
    curve: Curve.EaseInOut,
    duration: 200,
  }, () => { this.angle += 360; })
}
```

**依赖**：仅导入 `NavHeaderBar` from 'lib_common'

### TabItem 组件（CustomTabBar.ets 内部）
- **一码通Tab特殊处理**：
  - 判断 `this.index === TabBarType.ONE_CODE_PASS`
  - 已登录（isRelative）：显示48x48图标，position向上偏移30px，点击 pushPathByName 跳转
  - 未登录：显示 LoginButtonIcon 组件（FunctionalButton + GET_PHONE_NUMBER 授权），授权成功后回调跳转

## 3. 关键API使用

- [核心] `@kit.ArkUI` → `AppStorageV2.connect()`：连接全局状态（UserInfoModel、GlobalPointUtils）
- [核心] `@kit.ArkUI` → `QRCode(content)`：渲染二维码组件
- [核心] `@kit.ScenarioFusionKit` → `FunctionalButton`：华为账号手机号授权按钮
- [核心] `@kit.ScenarioFusionKit` → `functionalButtonComponentManager.OpenType.GET_PHONE_NUMBER`：授权类型
- [核心] `functionalButtonComponentManager.FunctionalButtonController().onGetPhoneNumber()`：授权回调
- [辅助] `@kit.ArkUI` → `getUIContext().animateTo()`：属性动画
- [辅助] `setInterval` / `clearInterval`：定时器管理

**第三方依赖**：
- `lib_common`：公共模块（NavHeaderBar、PageHeaderComp、RouterUtil、RouterMap、ContextUtils、UserInfoModel、TabBarType、LoginButtonIcon、LoginUtils、LoadingDialogUtil）
- `module_points`：积分模块（GlobalPointUtils）

## 4. 数据模型

```typescript
// UserInfoModel（来自 lib_common）
class UserInfoModel {
  isRelative: boolean;          // 是否已登录
  userInfo: {
    id: string;                 // 用户ID，用于生成二维码
    userTel: string;            // 用户手机号
  }
}

// GlobalPointUtils（来自 module_points）
class GlobalPointUtils {
  isCheckIn: boolean;           // 今日是否已签到
  myPoints: number;             // 当前积分
}

// TabBarType 枚举（来自 lib_common）
enum TabBarType {
  HOME_PAGE = 0,
  ONE_CODE_PASS = 1,           // 一码通Tab索引
  MINE = 2
}

// TabBarData 接口（来自 lib_common）
interface TabBarData {
  id: TabBarType;
  title: ResourceStr;
  activeIcon?: ResourceStr;
  defaultIcon?: ResourceStr;
}

// DiamondZone（来自 lib_common）- "我的"页面快捷入口
class DiamondZone {
  icon: ResourceStr;
  title: string;
  url: string;                  // 路由名称
}
```

## 5. 网络请求

无显式网络请求。所有数据来源于本地全局状态（AppStorageV2）和本地计算。

**华为账号授权（非网络请求，系统级能力）**：
- 授权方式：FunctionalButton + GET_PHONE_NUMBER
- 成功回调：`LoginUtils.onBindSuccess(data.code, callback)`
- 失败回调：`LoginUtils.onBindFail(callback)`
- 授权码（authCode）通过回调传递给 LoginUtils 处理

## 6. 路由配置

```json
// features/business_mine/src/main/resources/base/profile/route_map_mine.json
{
  "name": "MemberShipPage",
  "pageSourceFile": "src/main/ets/pages/MemberShipPage.ets",
  "buildFunction": "MemberShipPageBuilder"
},
{
  "name": "MemberShipManualPage",
  "pageSourceFile": "src/main/ets/pages/MemberShipManualPage.ets",
  "buildFunction": "MemberShipManualPageBuilder"
}
```

```typescript
// commons/lib_common/src/main/ets/constants/RouterMap.ets
export enum RouterMap {
  MEMBER_SHIP_PAGE = 'MemberShipPage',       // [核心] 一码通主页面路由
  MEMBER_SHIP_MANUAL = 'MemberShipManualPage', // [辅助] 会员手册页面路由
}
```

**导航方式**：
- Tab点击 → `RouterUtil.pushPathByName(RouterMap.MEMBER_SHIP_PAGE)`（push 模式）
- 会员手册入口 → `RouterUtil.pushPathByName(RouterMap.MEMBER_SHIP_MANUAL)`
